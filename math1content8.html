Exponential Functions and Geometric Sequences

Tutorial

Exponential function is a function where the base is a fixed number, while the power is the variable. One example is f(x)=2^x.

Geometric sequences are seuqences where each successive number is a fixed multiple of the number before it. One example is {2, 4, 8, 16, 32...} where the multiple is 2.

Rational exponents are exponents represent an interger exponent and an nth root. The root in the the denominator, while the exponent is in the numerator.
