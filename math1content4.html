Functions and Function notation

Tutorial

A function is an equation that will take an input, and give an output. Functions are written as f(x)=x+1; the x in f(x) in the input, and x+1 is the output. When evaluating functions, a value is given for x. An input can only have one output, but an output can be linked to multiple inputs.
