Systems of Equations

Tutorial

A system of equations is a set of equations that go together, with multiple variables. There are three ways to solve a system of equations: elimination, substitution, and graphing. Elimination is used when two equations have the same coefficient. In substitution, one variable is found indiviually, then subtituted to find the others. Graphing is self explanatory; you simply graph all of the equations in the system, then find the point where they intersect.

A system of linear inequalities is very similar to a system of equations, except it's a set of inequalities.
